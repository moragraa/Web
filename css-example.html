Pasos para actualizar de manera remota:
-Primero se hace el cambio en el original
-Se abre el source code
-Se agrega a staged changes el html
-Se guardan los cambio
-Se agregan los changes a staged changes para hacer un working tree
-Cuando se terminen de hacer todos los changes se escribe en message el nombre de los cambios
-Se abre la terminal en la carpeta donde esten
-Se escribe el codigo "git push origin master -f"
-Se actualiza de manera remota
